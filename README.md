# Modern Glassmorphism Calculator

A sleek, modern calculator application built with React, TypeScript, and Tailwind CSS. It features a responsive glassmorphism design, full keyboard support, and arithmetic history.

## Features

- **Modern UI**: Glassmorphism aesthetic with subtle gradients and shadows.
- **Keyboard Support**: Fully functional with physical keyboard inputs (Numpad support).
- **History**: Tracks previous calculations.
- **Responsive**: Works seamlessly on mobile and desktop.
- **Advanced Operations**: Includes Percentage, Negation, and standard arithmetic.

## Tech Stack

- **React**: UI Library
- **TypeScript**: Type safety
- **Tailwind CSS**: Styling
- **Lucide React**: Icons
- **Clsx & Tailwind-merge**: Class name management

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm start
   ```
